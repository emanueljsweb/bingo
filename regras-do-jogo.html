Objetivo do Jogo

O objetivo do bingo é ser o primeiro jogador a completar um padrão específico de números em sua cartela.

Regras do Jogo

  1. É necessário adicionar as cartelas, clicando em "Adicionar Cartela" e informando o nome do jogador.
  2. O jogo é conduzido por um sorteio de bolas numeradas. O sorteio é realizado por um sistema automático e os números vão sendo exibidos na tela.
  3. Conforme as bolas são sorteadas, o sistema automaticamente verifica as cartelas para ver se possuem o número sorteado.
  4. Caso tenha o número sorteado, ele é marcado automaticamente.
  5. O primeiro jogador que conseguir marcar todos os números necessários para completar um dos padrões aceitos, ganhará o jogo automaticamente.
  6. Após um jogador ser declarado vencedor, é possível clicar no botão "Jogar novamente" para reiniciar o jogo.
  7. Durante a realização de um jogo, não é possível adicionar ou remover cartelas.
  8. Conforme as bolas são sorteadas, os jogadores verificam suas cartelas para ver se possuem o número sorteado.

Padrões do Bingo

Existem diferentes padrões de bingo que podem ser utilizados para determinar a condição de vitória. Os padrões aceitos pelo Bingo On-line são:

  • Horizontal: Marcar todos os números de uma linha horizontal.
  • Vertical: Marcar todos os números de uma linha vertical.
  • Diagonal: Marcar todos os números de uma linha diagonal.
